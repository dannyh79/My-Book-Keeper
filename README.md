# My Book Keeper
A simple book keeper utilizing Rails 6 & Vue. Data is kept in persistent storage.

## To Install
1. Run `$ bundle install`
2. Run `$ rails db:migrate`

## To Start the Server
- Option 1.
  - Run `$ foreman start`

- Option 2.
  1. Run `$ rails s`
  2. Run `$ bin/webpack-dev-server`

## To Access the Page
Type `localhost:3000` in your browser's URL bar
